rygel_value_set_icon_info ()

void                rygel_value_set_icon_info           (GValue *value,
                                                         gpointer v_object);

Set the contents of a RYGEL_TYPE_ICON_INFO derived GValue to v_object.

rygel_value_set_icon_info() increases the reference count of v_object (the GValue holds a reference to v_object). If you do not wish to increase the reference count of the object (i.e. you wish to pass your current reference to the GValue because you no longer need it), use rygel_value_take_icon_info() instead.

It is important that your GValue holds a reference to v_object (either its own, or one it has taken) to ensure that the object won't be destroyed while the GValue still exists).

rygel_value_take_icon_info ()

void                rygel_value_take_icon_info          (GValue *value,
                                                         gpointer v_object);

Sets the contents of a RYGEL_TYPE_ICON_INFO derived GValue to v_object and takes over the ownership of the callers reference to v_object; the caller doesn't have to unref it any more (i.e. the reference count of the object is not increased).

If you want the GValue to hold its own reference to v_object, use rygel_value_set_icon_info() instead.
