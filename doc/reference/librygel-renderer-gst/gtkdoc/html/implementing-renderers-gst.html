Implementing GStreamer-based Renderers: Overview

This library may be used to create in-process UPnP/DLNA Renderers, or Players based on GStreamer, via the RygelPlaybinRenderer class. For instance, this may be used to add UPnP/DLNA Renderer or Player capability to an existing GStreamer-based media player application.

The RygelPlaybinRenderer reference documentation and these examples explain how to implement renderers.
